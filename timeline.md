# Project Timeline: Flood Change Captioning

## Phase 1: Literature Review (Week 1)
- Survey flood detection and change detection literature
- Study vision–language models
- Finalize problem statement

## Phase 2: Dataset Preparation (Week 2)
- Download Sentinel imagery
- Preprocess and align before–after pairs
- Create flood masks

## Phase 3: Change Detection Model (Week 3–4)
- Implement baseline change detection
- Fine-tune model for flood events
- Generate binary and multi-class change maps

## Phase 4: Caption Generation (Week 5–6)
- Extract visual features
- Integrate vision–language model
- Generate flood impact captions

## Phase 5: Experiments & Evaluation (Week 7)
- Quantitative metrics
- Qualitative analysis
- Ablation studies

## Phase 6: Paper Writing (Week 8–9)
- IEEE LaTeX paper drafting
- Figures and tables
- Review and revisions

## Phase 7: Finalization (Week 10)
- Code cleanup
- Final paper submission
